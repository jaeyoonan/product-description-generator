# WIREFRAME.md — 태안 녹용 상세페이지

> **타입**: 상품 상세 설명 이미지 (Product Description Long-form)
> **구조**: 부위 × 원산지별 개별 페이지 (4부위 × 2국가 = 총 8페이지)
> **레이아웃**: 수직 스크롤, 단일 컬럼
> **디자인 가이드**: [DESIGN.md](./DESIGN.md) 참조
> **최종 업데이트**: 2026-05-14

> **토큰 표기 규칙**:
> - `{token}` — 인라인 텍스트/값
> - `{[token]}` — 콘텐츠 영역 (텍스트 블록/이미지 영역 등)

## 페이지 매트릭스

각 상품 상세페이지는 **하나의 부위 + 하나의 원산지** 조합입니다.

| | 🇳🇿 뉴질랜드 | 🇷🇺 러시아 |
|---|---|---|
| **분골** (프리미엄 A) | `bungol-nz` | `bungol-ru` |
| **상대** (프리미엄 A) | `sangdae-nz` | `sangdae-ru` |
| **중대** (A등급) | `jungdae-nz` | `jungdae-ru` |
| **하대** (A등급) | `hadae-nz` | `hadae-ru` |

## 페이지 구조 (단일 부위 × 단일 원산지)

> 각 페이지는 아래 6개 섹션 순서로 구성됩니다.
> 표기법: `[ ]` = 이미지/미디어 영역, `---` = 구분선

| 순서 | 섹션 | 내용 | 배경 |
|:---:|------|------|------|
| 1 | 히어로 | 해당 부위 + 원산지 소개 | Parchment |
| 2 | 부위 상세 | 외향적 특징, 성분 명칭 | Surface |
| 3 | 부위별 비교표 | 4부위 비교 (현재 부위 강조) | Surface |
| 4 | 원산지 안내 | 원산지/파트너 스토리 | Parchment / Bark Split |
| 5 | 공통 — 품질·제조·보관 | 인증, 8단계 공정, 유의사항 | Surface / Parchment 교차 |
| 6 | 공통 — 법적 고지 | 의료용 한약재 안내 | Bark (다크) |
